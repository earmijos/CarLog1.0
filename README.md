# CarLog

CarLog is a simple vehicle log management app (Maintenance, Repairs, Fuel Log, Car Info).

## Backend
- Python + Flask
- SQLite database (`carlog.db`)

### Run Backend
```bash
cd backend
pip install -r requirements.txt or install flask
python app.py

## Frontend
- Flutter

### Run Frontend
```bash
cd frontend
flutter pub get
flutter run -d edge
